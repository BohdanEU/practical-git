# 4. Сформулювати функціональні сценарії (User Stories)

Функціональні сценарії описують взаємодію користувачів із системою. Вони формулюються у вигляді user stories, що дозволяє зрозуміти потреби користувачів та очікуваний результат використання системи.

User stories формулюються у форматі:
"Як [роль], я хочу [ціль], щоб [результат]".

## User Story 1

Як клієнт, я хочу переглядати каталог товарів, щоб знайти потрібний товар для покупки.

## User Story 2

Як клієнт, я хочу додавати товари до кошика, щоб оформити замовлення на декілька товарів одночасно.

## User Story 3

Як клієнт, я хочу оформлювати замовлення через сайт, щоб придбати обрані товари.

## User Story 4

Як адміністратор, я хочу додавати та редагувати товари в каталозі, щоб підтримувати актуальність інформації про товари.

## User Story 5

Як менеджер магазину, я хочу переглядати нові замовлення, щоб обробляти їх та організовувати доставку товарів клієнтам.

Функціональні сценарії дозволяють зрозуміти взаємодію користувачів із системою та визначити основні функції, які повинна реалізовувати інформаційна система.
